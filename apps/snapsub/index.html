SnapSub: スクショでサブスク管理

AIで自動登録！毎月の支出を把握

サブスク管理をもっとスマートに、もっと簡単に。SnapSubは、AIがスクリーンショットを自動解析し、煩わしい手入力を不要にします。月々の合計費用はもちろん、サービスごとの支出割合や過去の履歴もグラフで「見える化」。支払い忘れを防ぐ通知機能も搭載。賢くサブスクを管理し、無駄な出費を減らしましょう。

主な特徴

🤖

AIで超高速登録

スクショを撮るだけで、AIがサービス名・料金・サイクル等を自動抽出。月に15回までAI解析が0円。手動の追加・編集も簡単。

📊

支出を「見える化」

ホームで月間合計を確認。円グラフで割合、棒グラフで月別推移を表示。リストと連動して詳細も即チェック。

🔔

支払い忘れを防止

支払い期日を事前にローカル通知でお知らせ。通知タイミングは自由に設定できます。

🔒

安心のプライバシー

サブスク情報は端末内に保存。スクショは解析のためだけに利用し、当方が収集・保存することはありません。

スクリーンショット

スクショからサブスク情報を抽出
月額の合計を自動計算
支払いスケジュールをリマインド
カテゴリ別の支出割合を可視化

こんな方におすすめ

  • ✓ 音楽/動画配信やクラウド等、複数のサブスクを使っている方
  • ✓ 毎月のサブスク支出がいくらか把握したい方
  • ✓ 手軽に、正確にサブスク管理をしたい方
  • ✓ 無駄な支出を見直して節約したい方
  • ✓ 支払い日を忘れがちな方
